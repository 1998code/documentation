## Reinitializing the application 
Rebuild the administration panel with `YARN` or `NPM` <br>
Start the application with `YARN` or `NPM`

<code-group>

<code-block title="NPM">
```sh
npm run build
npm run develop
```
</code-block>

<code-block title="YARN">
```sh
yarn build
yarn develop
```
</code-block>

</code-group>
